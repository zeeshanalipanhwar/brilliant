Reversed Multiples

Computer Science Level 3

Let the reverse of a positive integer nnn, denoted R(n),R(n),R(n), be the result when the digits of the number are written backwards; for example, R(190)=091,R(190) = 091,R(190)=091, or just 91.91.91.

Call a positive integer nnn brilliant if n+R(n)n + R(n)n+R(n)

is a multiple of 13. Let BBB be the 100001000010000th brilliant number. Compute the last three digits of B.B.B.


The answer is 241.

C implementation without strings.

#include <stdio.h>
#include <inttypes.h>

uint64_t r(uint64_t x) {
    uint64_t ans = 0;   
    while (x > 0) {
        ans = ans*10 + (x%10);
        x /= 10;
    }   
    return ans;
}

int main(void) {
    uint64_t n = 0;
    for (int i = 10000; i > 0;) {
        n++;
        if ((n + r(n)) % 13 == 0) i--;
    }
    printf("the 10000th brilliant number: %" PRIu64 "\n", n);
    return 0;
}
