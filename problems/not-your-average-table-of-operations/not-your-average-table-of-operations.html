Not your average table of operations

Computer Science Level 4

Consider the 10-digit integer

N=ABCDEFGHIJ‾N=\overline { ABCDEFGHIJ} N=ABCDEFGHIJ

(where A,B,C,D..JA,B,C,D..JA,B,C,D..J are the digits of NNN). Each digit of NNN is a number between 000 and 999 inclusive and all digits of NNN are different from each other.

What must be the value of NNN so that the three vertical and three horizontal operations in the table below are correct?

EDJH‾÷JF‾=AA‾−++EDB‾×I‾=EHCG‾===EEID‾−DJ‾=EEAE‾\begin{matrix} \overline { EDJH } & \div & \overline { JF } & = & \overline { AA } \\ - & \quad & + & \quad & + \\ \overline { EDB } & \times & \overline { I } & = & \overline { EHCG } \\ \quad = & \quad & = & \quad & = \\ \overline { EEID } & - & \overline { DJ } & = & \overline { EEAE } \end{matrix}EDJH−EDB=EEID​÷×−​JF+I=DJ​===​AA+EHCG=EEAE​

Details and assumptions

You may assue that only one value of NNN satisfies the constraints above.


The answer is 5793146082.

A systematic approach is possible. For example, the units digit of EEID−DJ=EEAEEEID - DJ = EEAEEEID−DJ=EEAE tells us that J+E≡D(mod10)J + E \equiv D \pmod{10}J+E≡D(mod10). The tens digit of JF+I=DJJF + I = DJJF+I=DJ tells us that D=J+1D = J + 1D=J+1. Therefore, E=1E = 1E=1.

Then the hundreds digit of AA+EHCG=EEAEAA + EHCG = EEAEAA+EHCG=EEAE tells us that H+1=EH + 1 = EH+1=E, so H=0H = 0H=0. All the other digits can be worked out similarly.

from itertools import *
from operator import add
Map = {chr(i+65):i for i in range(10)}

def Get_Number(permutation,String):
    Number = ''
    for Letter in String:
        Number += permutation[Map[Letter]]
    return float(Number)

def Satisfies(permutation):
    p , I = permutation , Get_Number
    if I(p,'EDJH') / I(p,'JF') != I(p,'AA'):
        return False
    elif I(p,'EDB') * I(p,'I') != I(p,'EHCG'):
        return False
    elif I(p,'EEID') - I(p,'DJ') != I(p,'EEAE'):
        return False
    elif I(p,'EDJH') - I(p,'EDB') != I(p,'EEID'):
        return False
    elif I(p,'JF') + I(p,'I') != I(p,'DJ'):
        return False
    elif I(p,'AA') + I(p,'EHCG') != I(p,'EEAE'):
        return False
    print reduce(add,permutation)
    return True

for permutation in permutations('0123456789'):
    if permutation[0]=='0':continue
    if Satisfies(permutation):
        break

import itertools
def comb():
    s = '0123456789'
    for i in itertools.permutations(s, 10):
        if i[0] == '0':
            continue
        yield i
def concat(*args):
    return int("".join(args))

for i in comb():
    a, b, c, d, e, f, g, h, i, j = i
    if concat(e,d,j,h)/concat(j,f)==concat(a,a) and \
        concat(e,d,b)*concat(i)==concat(e,h,c,g) and \
        concat(e,e,i,d)-concat(d,j)==concat(e,e,a,e) and \
        concat(e,d,j,h)-concat(e,d,b)==concat(e,e,i,d) and \
        concat(j,f)+concat(i)==concat(d,j) and \
        concat(a,a)+concat(e,h,c,g)==concat(e,e,a,e):
        print concat(a, b, c, d, e, f, g, h, i, j)
        break
