3 coins

Calculus Level 3

The centers of three identical coins form the angle that's colored green above.

What angle maximizes the area of the blue convex hull?

Note: You can imagine the perimeter of the convex hull as a rubber band stretched around the three coins.

60 degrees 90 degrees 120 degrees 135 degrees 150 degrees 180 degrees

The white sectors together make a whole circle regardless of the angle. We can ignore them.

The orange rectangles also do not depend on the angle we can ignore them too.

Let the coins have radius 1 and the angle be 2θ2\theta2θ then the triangle has area 2sin⁡2θ2\sin{2\theta}2sin2θ and the blue rectangle has area 4sin⁡θ4\sin{\theta}4sinθ so the area as a function of θ\thetaθ is

A(θ)=2sin⁡2θ+4sin⁡θA(\theta)=2\sin{2\theta}+4\sin{\theta}A(θ)=2sin2θ+4sinθ

A′(θ)=4cos⁡2θ+4cos⁡θA'(\theta)=4\cos{2\theta}+4\cos{\theta}A′(θ)=4cos2θ+4cosθ

Solving A′(θ)=0A'(\theta)=0A′(θ)=0

4cos⁡2θ+4cos⁡θ=04\cos{2\theta}+4\cos{\theta}=04cos2θ+4cosθ=0

2cos⁡2θ+cos⁡θ−1=02\cos^{2}\theta+\cos{\theta}-1=02cos2θ+cosθ−1=0

cos⁡θ=12orcos⁡θ=−1\cos\theta = \frac{1}{2} or \cos\theta=-1cosθ=21​orcosθ=−1

θ=60or180\theta=60 or 180θ=60or180

2θ=120or3602\theta=120 or 3602θ=120or360

The solution that makes sense is 2θ=120degrees2\theta=\boxed{120 degrees}2θ=120degrees​

Note this picture is not the exact solution but it is close. The middle circle would have to be tangent to the line joining the other circles' centers.

I used the same approach, based on the insight that that only the rectangle-triangle region ever changed in area and that the three set of circle regions defined by the various lines each preserved a full circle's area regardless of angle, but used the algebraic expressions from the Pythagorean theorem, area as a function of the altitude of the triangle (ranging from 0 at 180 to √3 at 60) instead of the trig functions. Evaluating the derivative of that function (using unit circles) produced a maximum at h = 1 (and minimum at h = 0) corresponding to an angle of 120 degrees.

Relevant wiki: Optimization

The light and dark gray areas are constant, so we must maximize the blue area.

The blue area is the area of this pentagon minus the three green sectors. If we set the circle radii to be 1, the area of a sector is half the angle, so we have:

Agreen=θ+2(π−θ2)2=πA_{green} = \frac{\theta + 2(\pi-\frac{\theta}{2})}{2} = \pi Agreen​=2θ+2(π−2θ​)​=π

Therefore the problem reduces to maximizing the pentagon area, which is equivalent to maximizing one half as follows:

By basic trigonometry:

b=2cos⁡θ2a2+b2=4b=4−a2dbda=1214−a2(−2a)=−ab \begin{aligned} b &= 2 \cos\frac{\theta}{2} \\ a^2 + b^2 &= 4 \\ b &= \sqrt{4-a^2} \\ \dfrac{\text{d}b}{\text{d}a} &= \frac{1}{2}\frac{1}{\sqrt{4-a^2}}(-2a) = -\frac{a}{b} \\ \end{aligned} ba2+b2bdadb​​=2cos2θ​=4=4−a2​=21​4−a2​1​(−2a)=−ba​​

Now the area to maximize is the area of the rectangle plus the area of the triangle and the solution is where the gradient with respect to aaa is zero:

A=a+ab2=a(1+b2)dAda=0=1+b2+a(12dbda)=1+b2−a22b=1+b2−4−b22b=b+b22−2+b22=b2+b−2=(b−1)(b+2) \begin{aligned} A &= a + \frac{ab}{2} \\ &= a(1 + \frac{b}{2}) \\ \dfrac{\text{d}A}{\text{d}a} = 0 &= 1 + \frac{b}{2} + a(\frac{1}{2}\dfrac{\text{d}b}{\text{d}a}) \\ &= 1 + \frac{b}{2} - \frac{a^2}{2b} \\ &= 1 + \frac{b}{2} - \frac{4-b^2}{2b} \\ &= b + \frac{b^2}{2} - 2 + \frac{b^2}{2} \\ &= b^2 + b - 2 \\ &= (b-1)(b+2) \end{aligned} AdadA​=0​=a+2ab​=a(1+2b​)=1+2b​+a(21​dadb​)=1+2b​−2ba2​=1+2b​−2b4−b2​=b+2b2​−2+2b2​=b2+b−2=(b−1)(b+2)​

and since 0≤b≤20 \leq b \leq 20≤b≤2 we have b=1b = 1b=1. To confirm this is a maximum, we can easily show that the second derivative at b=1b=1b=1 is −33-3\sqrt{3}−33​.

We could now calculate θ\thetaθ using an arccos⁡\arccosarccos, however it's clear that the triangle above is the 1,2,31,2,\sqrt{3}1,2,3​ triangle making θ2=60∘\frac{\theta}{2} = 60^\circ2θ​=60∘ and θ=120∘\boxed{\theta = 120^\circ}θ=120∘​.

import megamu.mesh.*;

// points to hull up
float[][] points = new float[360*3][2];
int currentPoint = 0;

// radius of the cirlce
int r=100;

// the angle in the middle
float ang=120;

int inc=-1;

void setup() {
size(600,600);

}

int tmax=0;
float rmax=0;

void drawCircle(float x, float y, float r, int start, int stop) {
  for (int i=0; i < 360; i++) {
      float x2 = r * cos(radians(i));
      float y2 = r * sin(radians(i));

      //ellipse(x+x2, y+y2, 3,3);
      points[currentPoint][0] = x + x2;
      points[currentPoint][1] = y + y2;
      currentPoint++;
  }
}

void draw() {
currentPoint = 0;
background(0);
translate(width/2, height/2);

ang += inc;
if (ang <=60) { inc=1;}
if ( ang>=180) {inc=-1;}


// 180 degreees total, subtract the angle we have set, then divide by two to find
// the side angles 
float angSide = -1 * (180-ang)/2;

// Circle 1
//ellipse(0,0, r*2,r*2);
drawCircle(0,0,r, 45,135);

// Circle 2
float c2x = 2 * r * cos(radians(angSide));
float c2y = 2 * r * sin(radians(angSide));

//println( c2x,c2y);
//ellipse(c2x, c2y, r*2,r*2);
drawCircle(c2x,c2y,r,270,360);
line(0,0, c2x,c2y);

// Circle 3
float c3x = 2 * r * cos( radians( 180-angSide) );
float c3y = 2 * r * sin( radians( 180-angSide) );
//ellipse(c3x, c3y, r*2,r*2);
drawCircle(c3x,c3y,r, 180,270);
line(0,0, c3x,c3y);

Hull myHull = new Hull( points );

MPolygon myRegion = myHull.getRegion();
fill(255,0,0);
myRegion.draw(this);

int total=0;
for (int x=0; x< width; x++) {
  for (int y=0; y< height; y++) {
    //println(red(get(x,y)));
    if (  red(get(x,y)) == 255 ) {
       total = total + 1;
    }
  }
}

//println( ang, total);

if (total > tmax) { 
  tmax=total; 
  rmax=ang; 
  println("new max:", total, ang);
}
}
